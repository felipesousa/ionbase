# Ionbase

A simple integration with Ionic and Firebase! :gift: :fire: 

## Installation

1. Clone repository

```bash 
$ git clone git@github.com:felipesousa/ionbase.git newApp
```

## Usage

1. Install dependencies

```bash 
$ cd newApp && npm install
```

2. Execute gulp task

```bash 
$ gulp
```
